Deskripsi Singkat Aplikasi
Aplikasi ini adalah platform blog modern berbasis PHP Native dengan antarmuka dinamis menggunakan CSS kustom dan Tailwind CSS. Aplikasi terdiri dari dua bagian utama:
1. **Frontend Blog:** Beranda untuk menampilkan daftar artikel berdasarkan kategori (Pemrograman, Teknologi, Tutorial, Tips & Trick) serta halaman detail artikel.
2. **Backend CMS Admin:** Panel kontrol khusus admin/penulis untuk mengelola manajemen data penulis, artikel blog, dan kategori secara dinamis lengkap dengan fitur autentikasi session login dan logout.

Langkah-langkah Menjalankan Aplikasi Secara Lokal

1. Prasyarat (Prerequisites)
Sebelum menjalankan aplikasi, pastikan komputer Anda telah terinstal modul berikut:
* **XAMPP** (Minimal PHP Versi 7.4 atau versi di atasnya)
* **Web Browser** (Google Chrome, Microsoft Edge, atau sejenisnya)
* **Git** (Untuk kebutuhan manajemen repositori)
